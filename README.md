# DesktopBuddy.exe
is a Psychological horror game of the same category of games like Doki-Doki Literature Club, and KinitoPET. The game is still in development and is not for public release. If you'd like a demo of the desktop, then visit [this Codetorch page](https://codetorch.net/en/projects/91273).

The actual game is about a desktop friend who can rewrite his own code via his very own Advanced Programming Integration (API), and can adapt to what you need——his goal is to be the BEST friend possible.
